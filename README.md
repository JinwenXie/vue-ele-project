# vue-ele-project

#### 环境参数

- 环境配置：不限 
- CSS预处理： Stylus 
- JS： ES6 
- 代码风格检查工具： ESLint 
- Node.js： v8.11.0+ 
- Vue.js： v2.5.17+ 
- Vue-cli： v3.0+ 
- 组件库： cube-ui v1.12.3
	

#### 执行步骤

	cd vue-ele-project/code // 进入文件

	npm install // 安装依赖
	
	npm run serve // 开发环境编译

##### 生产环境编译

	npm run build
